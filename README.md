## ● Controls:
• A / ← / [Left Touch Button] - Move left (in game)<br>
• D / → / [Right Touch Button] - Move right (in game)<br>
• W / ↑ - Move up (in menu)<br>
• S / ↓ - Move down (in menu)<br>
• Mouse Movement - Navigate buttons (in menu / in game over)<br>
• Spacebar / Mouse Button - Confirm (in menu / in game over)

## ● Score System:
• Apple = 1 point<br>
• Banana = 2 points<br>
• Grape = 3 points<br>
• Melon = 4 points<br>
• Maxim Tomato = 5 points<br>
• Bomb = -7 points (if not invincible)<br>
• Poison Bomb = -1 point 10 times (if not invincible / does not kill)<br>
• Critical Bomb = Game Over (if not invincible)<br>
• Invincible Candy = Invincibility<br>
• Jumbo Candy = Invincibility + Bigger Size
